Getting started 🚀

Wie mach ich mit? Genau so:

  1. Element (Riot) App installieren
    Wähle die für dich passende Plattform.

  2. Benutzerkonto anlegen.
    homeserver = matrix.org

  3. Anmeldung verifizieren
    E-Mail öffnen und Link klickern

  4. Einloggern
    Username oder E-Mail + Passwort eingeben

  5. erste Nachricht schreiben an [ @danielito:matrix.org ]
    oder direkt über [ #trnbtl-invite:matrix.org ]

  6. Chat - Train - Sleep - Repeat
